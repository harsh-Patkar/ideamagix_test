Healthcare Website

Project Overview

This is a responsive healthcare website for Total Knee Replacement services. The website provides information about knee replacement surgery including treatment details, eligibility criteria, preparation steps, and recovery guidance.

Technologies Used

HTML5
CSS3
JavaScript
jQuery 3.6.0
Owl Carousel 2.3.4
Font Awesome 6.4.0

Project Structure

index.html - Main HTML file
styles.css - Stylesheet
script.js - JavaScript functionality
README.txt - This file
images folder - Contains all images

Features

Responsive design for all devices
Hero image slider
Navigation menu
Treatment information sections
Patient testimonials
Doctor profiles
Hospital listings
FAQ section
Contact form

How to Run

Open index.html in any web browser to view the website.

Browser Compatibility

Tested on Chrome, Firefox, Edge, Opera, and Safari.

Deployment

This website is deployed on Netlify. To deploy updates, simply drag and drop the project folder to Netlify dashboard or push changes to the connected repository.
